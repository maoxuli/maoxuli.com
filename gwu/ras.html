Realtor's Assistant System

Jan 2010 - Apr 2010, The George Washington University, Washington DC

This is the hands-on work for my class of Software Engineering at The George Washington University. This is a team-based project through which we practice and experience a complete software cycle. The work emphasized on the software processes: choosing appropriate development cycle model and team organization, making plans and tracking on the progress, moving forward from requirements phase to analysis, design, implementation and testing phase.

Please find code and documents at Google Code:

  • http://code.google.com/p/realtor-assistant/

Here is a simple demonstration of the system:

  • http://www.maoxuli.com/gwu/ras

Technical Summary

Languages and marks: PHP, HTML/CSS, JavaScript
Platform: LAMP (Linux, Apache, MySQL, PHP)
PHP framework: SpeedPHP (MVC)
