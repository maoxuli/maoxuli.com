DNS Resolver: Request and parse DNS resource records

Jan 2011 - Apr 2011, The George Washington University, Washington DC

This is the hands-on work for my class at The George Washington University. I choose DNS resolver as a practice of programming with network protocols. A set of C++ classes are designed to wrap the procedure of requesting resource records from DNS server and parsing response message with resource records. The architecture is easy to extend to support new formats of resource records.

A request packet is sent to DNS server, then the result is returned in a response packet. Current implementation only supports sending message with UDP protocol, and only parses resource records with types of A, CNAME, and TXT, although other types of resource records are also returned with response packet.
