[ GWU | MParser ]

Case study: A media file parser framework

Jan 2011 - Apr 2011, The George Washington University, Washington DC

This is a hands-on practice for my coursework at The George Washington University. Media file parser is taken as a case study of design patterns and software refactoring. ASF/WMV and RM/RMVB formats are analyzed and a software framework of media file parser is designed. With design patterns like strategy, factory, and facade, the framework is extendable to support parsing of new media formats.

Please find source code and documents at:
https://github.com/mparser
