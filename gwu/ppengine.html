A local media streaming service framework

Jan 2010 - Dec 2010, The George Washington University, Washington DC

This is a hands-on practice for my coursework at The George Washington University. RTSP/RTP is a widely used media streaming protocol. In this case, I design a local media streaming service framework based on refactoring of a peer-to-peer video streaming system. As a part of client-side application, the local media streaming service manages media buffering according to user's operations and push media data to a media decoder in a streaming fashion with RTSP and RTP protocols. RTSP is s HTTP extension. A wrapper class of HTTP is implemented first, which can be work as a HTTP server or a HTTP client. Then, a RTSP server is implemeted as a derived class of HTTP server. In addition, the implementation of thread, socket, and HTTP wrappers are cross platform over Mac OS/X and Windows.

Please find source code and documents at Github:
