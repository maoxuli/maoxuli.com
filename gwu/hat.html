Hat: A simple management system for personal photos

Jan 2011 - Apr 2011, The George Washington University, Washington DC

This is the hands-on work for my class at The George Washington University. I supposed that personal photos are scattered on heterogenous storage, such as desktop computers, laptops, mobile devices, and online services. We need to list and search on these photos, delete duplications on same storage, and make duplications on different storage for the purpose of backup. Hat defined a mixed system architecture to manage personal photos. A simple central management system is deployed to collect characteristics of the photos on heterogenous storage, and provide services like searching and detecting of identical or similar photos.

Some kinds of characteristic data of photos, such as file stat, meta data, hash, and histogram data, are extracted or calculated, and then collected into the central database. Functions like complex searching and finding identical or similar photos are realized based on database query and analysis of characteristic data. Other advanced functions then may be possible, such as managing personal photos on Facebook and Flickr, automatic deduplication and duplication of photos for backup, and so on.

The concept is demonstrated with a prototype design of software architecture and three simple command line programs. One is used for database update, data query, and analysis. One is used for monitoring local file system, detecting changes of photo files, extracting photo's characteristic data, updating records in database. The other is a user interface program. The programs are developed on Mac OS X system with standard C++. Three programs communicate to each other based on Zeroc Ice, a Internet communication middle-ware. Other libraries including OpenCV, ImageMagik, and OTL are used to simplify the development.

Technical Summary

Languages and libraries: C++, STL, SQL, OTL
Operating system and database: Mac OS X 10.6, SQLite 3
Edit, Compile, and Debug: Xcode 4
Network/ORB framework: Zero Ice 3.3
Image processing libraries: OpenCV, ImageMagik
Characteristic data analysis File stat, Meta data, Hash, Histogram, Euclidean distance
