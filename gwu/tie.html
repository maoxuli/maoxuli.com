Architecture: A simple social networking system for sharing moods

Sep 2010 - Dec 2010, The George Washington University, Washington DC

This is a hands-on practice for my coursework at The George Washington University. In this case study, I explored the typical architectures of social networking systems and designed a simple social networking system for sharing our moods with friends. The server side of the system is a web-based application running on LAMP platform (Linux, Apache, MySQL, PHP). The client side is a GUI application running on mobile devices (Android system). Server provides a web service interface to clients. Clients access server with HTTP protocol.

A proof-of-concept prototype is demonstrated in this case.
